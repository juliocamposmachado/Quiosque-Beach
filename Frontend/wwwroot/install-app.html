Instalação no Android

  1. Baixe o arquivo APK clicando no botão "Baixar APK"
  2. Nas configurações do seu dispositivo, habilite "Fontes desconhecidas"
  3. Localize o arquivo baixado e toque para instalar
  4. Siga as instruções na tela
